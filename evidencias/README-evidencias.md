# Evidências de Execução

Esta pasta armazena prints e resultados das execuções da coleção.

## Como exportar evidências do Postman

1. Execute a coleção completa clicando em **Run collection**
2. Ao finalizar, clique em **Export Results**
3. Salve o arquivo `.json` nesta pasta
4. Adicione prints da tela de resultados se desejar

## Resultado esperado

Todos os 8 testes devem passar com status ✅ verde.
